## 指针
概念:指向地址的变量(cs50里那个短片深入人心),*表示寻址,&表示取地址

常见表示地址的东东:
  - 数组名
  - 函数名
  - 字符串字面量 eg."HELLO"
  - 引用:起别名,底层是受限的指针 eg.int &a = b;
  
eg
- 指针数组 (Array of Pointers)
 int *p[5] 
 从 p 开始，向右结合 [] 成为数组，再向左结合 * 说明成员是指针。本质是一个装了 5 个指针的容器。

- 数组指针 (Pointer to an Array) 
  int (*p)[5] 
  括号改变优先级，p 先与 * 结合成为指针，再向右指向一个含 5 个元素的数组。本质是一个指向固定大小连续内存的箭头。

-  指向常量的指针 (Pointer to Constant) 
  const int *p 
  从 p 开始，向左结合 * 成为指针，再向左看到 const 修饰 int。
  内容只读，不能通过该指针修改所指的数据。

-  指针常量 (Constant Pointer) 
  int * const p 
  从 p 开始，向左先结合 const 修饰指针本身，再看到 *。地址只读，指针一旦绑定某个地址，不能再指向其他地方。

-  函数指针 (Function Pointer) 
  void (*f)(int) f 先与 * 结合成为指针，右侧 (int) 说明其指向的是函数。本质是存储了函数在代码段的入口地址。

-  指针的指针（二级指针）
   int **pp 
   从 pp 开始，向左连续结合两个 *。表示指向指针的指针，常用于动态二维数组或在函数内部修改外部指针的指向。
